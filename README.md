# ☕ Coffee Shop Sales Dashboard (Excel Project)

## 📌 Project Overview
This project focuses on analyzing sales performance, customer behavior, and product trends of a coffee shop using Microsoft Excel.

The dashboard helps management monitor key performance indicators and make data-driven decisions.

---

## 🎯 Business Problem Statement
The coffee shop management wants to:

- Monitor overall sales performance  
- Identify peak sales hours  
- Understand product contribution to revenue  
- Compare performance across store locations  
- Analyze weekday and size-based customer behavior  

This dashboard provides a centralized view to support decision-making.

---

## 📊 Key KPIs

- 💰 Total Sales  
- 👥 Total Footfall  
- 🧾 Average Bill per Person  
- 📦 Average Orders per Person  

---

## 🔍 Insights Delivered

- Identified peak sales hours (highest activity between 8 AM – 10 AM)
- Found top 5 revenue-generating products
- Compared footfall across Astoria, Hell’s Kitchen, and Lower Manhattan
- Evaluated weekday performance trends
- Analyzed product size distribution patterns
