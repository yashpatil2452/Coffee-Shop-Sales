Coffee Shop Sales Dashboard

Objective:
    -To analyze sales performance, customer behavior, and product trends of a coffee shop using Microsoft Excel.

Key KPIs:
    -Total Sales
    -Total Footfall
    -Average Bill per Person
    -Average Orders per Person

Business Insights Delivered:
    -Identified peak sales hours
    -Analyzed product-wise sales contribution
    -Compared footfall across locations
    -Evaluated weekday sales performance
    -Analyzed size distribution trends

Tools Used:
    -Microsoft Excel
    -Pivot Tables
    -Power Pivot
    -Slicers
    -Charts & KPI Cards
    -Data Cleaning & Modeling
